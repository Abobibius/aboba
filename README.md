TCP Chat Server (WinSock2)

Описание
Простой TCP-сервер на C++ под Windows, использующий WinSock2.
Сервер принимает одного клиента, обменивается текстовыми сообщениями
и поддерживает команды, начинающиеся с символа '/'.

Требования
- Windows
- Visual Studio (MSVC) или MinGW
- Библиотека ws2_32

Сборка (Visual Studio)
1. Создать проект Console Application (C++).
2. Вставить код сервера в main.cpp.
3. Собрать и запустить проект.

Сборка через командную строку (MSVC)
cl /EHsc server.cpp ws2_32.lib
server.exe

Как работает
- Сервер запускается и начинает слушать порт 8080.
- После подключения клиента начинается обмен сообщениями.
- Каждое сообщение передаётся в формате:
  [4 байта — длина сообщения][текст сообщения]

Использование
- Ввод в консоли сервера отправляется клиенту.
- Сообщения клиента выводятся в консоль сервера.

Команды
/help
Выводит список доступных команд и отправляет его клиенту.

/exit
Завершает работу сервера.

Переподключение
Если клиент отключается, сервер закрывает сокеты и
пытается заново принять подключение.

Настройки
- Порт по умолчанию: 8080
(изменяется в коде в функции createAndBind)

Замечания
- .cpp файл нельзя запустить напрямую — он должен быть скомпилирован.
- Рекомендуется добавить проверку на пустые сообщения.
- Желательно реализовать sendAll / recvAll для надёжной передачи данных.
